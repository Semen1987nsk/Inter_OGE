## Формула расчета

**F = mg** где:
- m = масса в кг
- g = 9.8 м/с² (ускорение свободного падения)

**Δl = F/k** где:
- F = сила тяжести в Н
- k = жесткость пружины в Н/м
- Δl = удлинение в м

**k = F/Δl** (проверочная формула)

## Критическая ошибка, которая была исправлена

**Проблема:** В функции `attachWeight` строка 2343:
```javascript
const elongationCm = elongationPx / this.physics.pixelsPerCm;  // ❌ НЕПРАВИЛЬНО
```

**Исправление:**
```javascript
const elongationCm = elongationM * 100;  // ✅ ПРАВИЛЬНО
```

**Объяснение:**
- `elongationM` уже в метрах (результат F/k)
- `elongationPx = elongationM * 100 * pixelsPerCm` (перевод в пиксели)
- Для получения сантиметров: `elongationCm = elongationM * 100`
